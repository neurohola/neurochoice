# NeuroChoice Website (Static)

## Files
- index.html
- styles.css
- images/adv.jpg
- images/products.jpg

## Run locally
Option 1: double-click index.html

Option 2 (recommended):
python -m http.server 8080
then open http://localhost:8080

## Deploy
Upload the folder contents to your hosting for neurochice.co.uk (or your domain).
